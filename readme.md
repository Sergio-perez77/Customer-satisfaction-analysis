Customer Stisfaction Analysis (CSAT)

*Project Overview/Descripcion del Proyecto:

-This project analyzes customer satisfaction (CSAT) using customer support data.

--Este proyecto analiza la satisfaccion del cliente (CSAT) utilizando datos de soporte.

*Tools used/Herramientas utilizadas:
-Excel (limpieza de datos)
-SQL
-Tableau

*Key Insights / Principales conclusiones:

-Most customers report high satisfaction (ratings 4-5).
-No strong correlation between handling time and satisfaction.
-Differences detected between channels and agents.

-La mayoria de los clientes presentan alta satisfaccion (valores4-5).
-No se observa una correlacion fuerte entre el tiempo de atencion y la satisfaccion.
-Se detectan diferencias entre canales y agentes.
